Projects

Statistical Analysis of the US Wildfires

Implemented different stages of the Data Science Pipeline namely EDA, feature engineering, data imputation, and ensemble modeling to analyze 24 years of US wildfire data. Trained a Gradient Boosting Classifier using sklearn to predict the cause of the fire (1/12 classification) with accuracy of 62% and size of the fire (1/7 classification) with an accuracy of 75%

Deep Q Network to play 2048

Implemented Deep Q Learning algorithm to train a reinforcement learning agent to play 2048 game. Tested the agent with different rewarding strategies and was able to achieve a best score of 512

Zero-Shot Learning: Face Verification using Transfer Learning

Created a data loader to generate positive and negative examples by concatenating two images of the same person and two images of different people, respectively. Modified the ResNet-18 model to train on these examples to classify if the faces are of same or different persons. Achieved an accuracy of 92% on test examples with previously unseen people in 15 epochs of training

Multiple Choice Question Answering by Fine-Tuning BERT

Pretrained BERT model on the CommonSenseQA dataset and fine-tuned on the OpenBookQA dataset. Attained accuracies of 71% and 58% on the OpenBookQA with and without relevant fact respectively

Language Modeling Using LSTM

Generated 100-dimensional word embeddings for ~33k tokens from the wikitext-2 dataset using a 2-layer LSTM network. Achieved batch perplexities of 205 and 316 on the train and test corpora, respectively with 20 epochs

Real-Time Activity recognition using DNN

Collected acceleration and gyro data for 12 different physical activities using IMU. Trained an autoencoder to detect anomalous activity other than 12 activities. Trained a deep neural network model in TensorFlow to detect the physical activity. Deployed the model on arduino nano 33 BLE sense for real time activity detection using Tensorflow Lite with an accuracy of 95%

Text classification on the state of the union speeches by US presidents

Implemented Naive Bayes model with expectation maximization to classify the ideology of Presidents prior to 1964 to one of the two modern day political ideologies in the US using state of the union speeches

Frame-based Information Extraction from Tweet Data

Implemented an NLP system that can extract award names, winners, presenters and nominees from Golden Globes awards ceremony using python libraries like RegEx, NLTK and Spacy. Achieved 78% completeness of total available information in the data with 86% accuracy

Classification of Images from Fingers Dataset

Implemented a 50-layer residual CNN model using identity blocks and convolutional blocks to train on and predict numbers shown using fingers on a grey scale image, with test accuracy of 87%

Date Translation Using an Attention Model

Implemented a sequence model using bidirectional LSTM layers as pre-attention layers and unidirectional LSTM layers as post-attention layers to translate human readable dates text to machine readable dates

Morse code detector project

Designed a Morse code detector system and implemented it on Basys 2 FPGA board. Explored and utilized Verilog using Xilinx ISE Design Suite and Finite State Machines. Morse code can be input in to the FPGA using push buttons over it, which maps the code into corresponding characters or alphabet. The characters are displayed on a seven-segment display present on FPGA board

Realtime spatial gait characterization

Implemented the Pedestrian Dead Reckoning for Spatial Gait Characterization. Analyzed PDR algorithm and implemented using Arduino for real time spatial gait characterization. Measured the spatial gait parameters in real time while a person is walking. Designed a portable Inertial sensing unit using IMU and Arduino Nano
